What is Freeware

Freeware is software that is released to the general public for no cost and it will not require any payment to use during it's entire lifespan. Freeware can have many different licenses, each affecting the end users ability to share or change the software differently. Keep in mind that the software is still the authors intellectual property and it is generally illegal to claim it as your own and/or make money off of it unless the author specifies otherwise.
It tends to be open source, which means it's source code is viewable and shared online.

Pros of Freeware:

  • Can make money from donations for creator
  • Rewarding to maintain if you have the time

Cons:

  • Motivation can be low, especially without income
  • Managing community involvement
  • Vetting for security issues, especially from pull requests

It can also compress into many formats including it's own .7z extension. While it is free, you can lose some convience options that you would have with paid software like WinRAR.
